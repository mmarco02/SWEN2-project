# SWEN2-project

## Für Development:

Datenbank starten:<br>
``docker-compose up db -d ``


Backend Starten:
``Swen2TourPlannerApplication``

Frontend Starten:
```
cd /frontend
npm ci
npm run dev
```


## Für alles ausführen

```
docker-compose up
```

führt datenbank, backend und frontent aus (aber dann kein hot-reload)
